# Extensive-Domains
Extensive Domains is a plugin that aims to bring aspects of diplomacy, economics, and geopolitics to Minecraft. Many of the features brought by it can be toggled on/off to suit any server's needs. The main goal Extensive Domains tries to achieve is customization from both players and server admins, granting them many opportunities to express themselves through the plugin's many features.

# Overview
Below are the main features provided by the plugin
* Creation of governments
* Creation of currencies
* Trading between governments
* Diplomacy
* Creation of laws
* Creation of jobs and work orders
* Global event/incident system
* Creation of cultures
* Exploitation of natural resources
* Creation of usable structures
* Quests system
